# Build a real-time chat app

A simple but complete real-time chat application.

## Features
- Users pick a username on first visit
- Multiple chat rooms (general, random, tech)
- Messages appear in real-time (use WebSocket)
- Show who's online in each room
- Message history persists (use SQLite)
- Typing indicators

## Tech
- Backend: Python FastAPI + WebSocket
- Frontend: vanilla HTML/CSS/JS (no framework)
- Database: SQLite
- No external services, everything runs locally

## Design
- Clean, modern UI (dark theme)
- Mobile responsive
- Smooth animations for new messages
