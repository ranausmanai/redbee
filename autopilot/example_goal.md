# Goal: Promote AutoShip

AutoShip is a CLI tool that turns a markdown spec into a running application. You give it a product description and it builds the entire thing autonomously using Claude or Codex.

## What to promote

- It's a single Python file (~60 lines)
- Zero dependencies, just stdlib
- Works with Claude CLI or Codex CLI
- Give it a spec, get a working app
- GitHub: https://github.com/ranausmanai/autoship

## Target audience

- Developers who want to prototype fast
- AI/LLM enthusiasts
- Indie hackers and side project builders
- People interested in AI-assisted development

Nothing yet — this is the initial launch push.
